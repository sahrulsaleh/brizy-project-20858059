Bimtek Teknik Membangun Web Server Linux Berbasis Cloud Computing Untuk Pemula

Pelatihan membangun server berbasis linux untuk beragam aplikasi website dan sistem informasi lainnya yang dapat diakses secara online maupun di dalam jaringan kantor/instansi

Pendahuluan

Komputer Server merupakan salah satu komponen vital bagi Organisasi yang sedang menggunakan sistem informasi. Memiliki pemahaman tentang server merupakan hal yang krusial bagi SDM instansi. Namun sayangnya pengetahuan tentang server masih dianggap rumit padahal semakin mudah untuk dipelajari.


Beberapa organisasi yang telah memiliki server, belum menggunakannya secara optimal seperti menempatkan satu aplikasi ringan dalam satu server spesifikasi tinggi. Di sisi lain, organisasi yang servernya mengalami overload ingin upgrade server namun belum mengetahui bagaimana teknik/metode yang paling efisien untuk melakukannya.


Dalam pelatihan ini, peserta akan diajarkan langsung cara membangun server mulai dari awal hingga sistem informasi tampil online sempurna berikut pengembangan servernya. Peserta juga dipandu agar paham tentang linux, perintah dasar linux, dan struktur web server sehingga ketika menemui kendala kedepannya, peserta dapat menganalisa penyebabnya secara mandiri.

Tujuan dan Manfaat

  1. Peserta mampu membangun dan mengelola server secara mandiri
  2. Organisasi memiliki tim SDM yang mampu membangun dan mengelola server
  3. Organisasi dapat mengoperasikan server dengan optimal dan hemat biaya
  4. Organisasi mampu melakukan upgrade dan downgrade kapasitas server

Materi Pelatihan

Pengenalan Cloud Computing

Membahas tentang keamanan, kemudahan, fleksibilitas teknologi cloud sekaligus regulasi perundang-undangan yang terkait

Mengenal Sistem Operasi Linux

Mengenal Sistem Operasi Linux mencakup struktur sistem dan perintah dasar yang sering digunakan dalam mengelola server

Instalasi Web Server

Instalasi web server, database, dan PHP sekaligus memahami struktur konfigurasi webserver

Manajemen DNS, CDN, Proxy, Email Forwarding

Mengelola DNS sebagai pusat pengaturan Alamat IP server, Distribusi konten melalui Content Delivery Network agar akses lebih cepat, Proxy Server (menyembunyikan alamat server original agar lebih aman), dan Email Forwarding

Instalasi Sistem Informasi

Memasang 1 sistem informasi ke server dilanjutkan beberapa sistem informasi lainnya ke server yang sama (multi sistem dalam 1 server)

Upgrade Server Model Remote Database

Praktikum Instalasi web server dengan server database terpisah (satu server untuk aplikasi + 1 server untuk database)

Upgrade Server Model Load Balancing

Praktikum Instalasi multi web server dengan konfigurasi (satu server proxy, dua server aplikasi, dan satu server database)

Tanya Jawab, Studi Kasus, Migrasi Sistem, Keamanan Server
